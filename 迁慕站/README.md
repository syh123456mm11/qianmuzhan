# 迁慕站 - 安全文件管理平台

一个现代化的安全文件管理平台，采用精美的UI设计和流畅的交互动画。

## 功能特性

### 隐蔽的管理入口
- 网站默认显示文件浏览页面（正规文件分享站）
- 连续点击Logo 10次触发密码验证（隐蔽的后台入口）
- 密码：`syh123456mm`
- 安全警报机制（连续输错3次触发）
- 2小时自动登出

### 文件管理功能
- 支持多格式文件上传：ZIP, RAR, EXE, ARJ, PY, TXT, PDF, DOC, XLS, 图片等
- 实时文件列表更新
- 文件下载统计
- 拖拽上传支持
- 搜索和排序功能

### 访客浏览系统
- 无需登录即可浏览下载所有文件
- 文件搜索和排序功能
- 文件预览信息（大小、上传时间、下载次数）

### UI设计特色
- 现代化渐变配色方案
- 流畅的交互动画效果
- 响应式设计，支持移动端
- 精心设计的图标系统
- 专业、正规的网站外观

## 使用说明

### 设置Logo
将您的PNG logo文件放入项目目录，命名为 `logo.png`

### 作为访客浏览文件
1. 直接打开 `index.html`
2. 浏览和下载公开文件
3. 使用搜索和排序功能查找文件

### 访问后台管理（隐蔽入口）
1. 打开 `index.html`
2. **连续点击顶部Logo 10次**（隐蔽操作，无视觉提示）
3. 输入密码：`syh123456mm`
4. 进入后台管理界面

### 管理后台功能

#### 📊 仪表盘
- 查看总文件数、总大小、今日上传数量
- 查看最近上传的10个文件
- 实时统计信息

#### 📁 文件上传
- **拖拽上传**：将文件拖拽到上传区域
- **点击上传**：点击"选择文件"按钮选择文件
- 支持格式：ZIP, RAR, EXE, ARJ, PY, TXT, PDF, DOC, XLS, 图片等
- 文件大小限制：100MB
- 上传进度显示

#### 📂 文件管理
- 查看所有文件的详细信息（名称、大小、上传时间、下载次数）
- **下载文件**：点击"下载"按钮
- **删除文件**：点击"删除"按钮（需要确认）
- 文件列表按上传时间倒序排列

#### ⚙️ 系统设置
- **修改密码**：更改管理员登录密码
- **自动登出**：2小时后自动退出（可配置）
- **存储信息**：查看存储路径和文件保留策略
- **清除数据**：一键清除所有上传的文件（谨慎使用）

#### 🌐 返回前台
- 点击导航菜单的"返回前台"
- 或者点击"退出登录"按钮
- 自动回到文件浏览界面

## 自定义配置

```javascript
this.CORRECT_PASSWORD = 'syh123456mm';
```

### 调整点击次数
在index.html的JavaScript中修改：
```javascript
this.TARGET_CLICKS = 10;
```

### 更改主题色
在 `styles.css` 中修改CSS变量：
```css
--primary-color: #6366f1;
--secondary-color: #8b5cf6;
```
